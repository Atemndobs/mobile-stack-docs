## Overview

Mobile Stack includes optional Text-to-Speech (TTS) capabilities that allow your app to convert text into spoken audio with multiple voice options.

<Note>
  TTS is an **optional feature** - enable it during app generation when prompted.
</Note>

## Available Voices

The TTS service includes multiple voice options:

```typescript
const VOICES = [
  { id: 'alloy', name: 'Alloy' },
  { id: 'echo', name: 'Echo' },
  { id: 'fable', name: 'Fable' },
  { id: 'onyx', name: 'Onyx' },
  { id: 'nova', name: 'Nova' },
  { id: 'shimmer', name: 'Shimmer' },
];
```

## Using TTS Service

```typescript
import { ttsService } from '@/services/tts';

// Save selected voice
await ttsService.saveSelectedVoice('alloy');

// Get selected voice
const voice = await ttsService.getSelectedVoice();

// Convert text to speech
const audioUrl = await generateSpeech('Hello world!', voice);
```

## Voice Dropdown Component

The generated app includes a `VoiceDropdown.tsx` component:

```typescript
import { VoiceDropdown } from '@/components/VoiceDropdown';

<VoiceDropdown
  selectedVoice={currentVoice}
  onSelectVoice={(voice) => setCurrentVoice(voice)}
/>
```

## Implementation Example

```typescript
import { useState } from 'react';
import { ttsService } from '@/services/tts';

export default function TTSScreen() {
  const [text, setText] = useState('');
  const [voice, setVoice] = useState('alloy');

  const speak = async () => {
    const audioUrl = await generateSpeech(text, voice);
    // Play audio using expo-av or similar
  };

  return (
    <View>
      <TextInput
        value={text}
        onChangeText={setText}
        placeholder="Enter text to speak"
      />
      <VoiceDropdown
        selectedVoice={voice}
        onSelectVoice={setVoice}
      />
      <Button title="Speak" onPress={speak} />
    </View>
  );
}
```
