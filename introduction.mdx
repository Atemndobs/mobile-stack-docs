## What is Mobile Stack?

Mobile Stack is a powerful CLI tool that generates full-featured React Native mobile applications with pre-configured authentication, database, AI integration, and analytics. Built by J&A Business Solutions for rapid mobile app development.

## Key Features

<AccordionGroup>
  <Accordion icon="shield-check" title="Authentication">
    Google OAuth via Supabase with protected routes and auth context provider out of the box.
  </Accordion>

  <Accordion icon="database" title="Database Ready">
    Supabase PostgreSQL with app-prefixed tables, row-level security, and migration templates.
  </Accordion>

  <Accordion icon="brain" title="AI Integration">
    Support for OpenAI (GPT-4o), Google Gemini (2.0 Flash), and Ollama with easy model switching.
  </Accordion>

  <Accordion icon="palette" title="Dynamic Theming">
    Beautiful light/dark mode system that works perfectly across all components.
  </Accordion>

  <Accordion icon="chart-line" title="Analytics">
    PostHog integration with event tracking utilities and pre-defined event constants.
  </Accordion>

  <Accordion icon="code" title="Type Safe">
    Full TypeScript support with strict typing and comprehensive interfaces.
  </Accordion>
</AccordionGroup>

## Quick Example

```bash
# Install globally
npm install -g @jnabusiness/mobile-stack

# Create a new app
mobile-stack create my-awesome-app

# Navigate to your app
cd my-awesome-app

# Install dependencies
npm install

# Start developing!
npm start
```

  Generated apps include comprehensive README files with detailed setup instructions, project structure explanation, and customization examples.
